# Quiz King Challenge (クイズ王チャレンジ) 👑

**Quiz King Challenge** は、GoogleのGeminiモデルを搭載したAIの対戦相手「クイズ王」と知識を競う、インタラクティブなトリビアゲームです。

このアプリケーションは、選択されたカテゴリに基づいてリアルタイムでクイズを生成し、ユーザーと一緒に回答するAIの対戦相手をシミュレートして、競争力のある魅力的な体験を提供します。

## ✨ 特徴

- **ダイナミックなクイズ生成**: 二度と同じクイズをプレイすることはありません。Gemini APIを使用して、選択したカテゴリに基づいて質問がリアルタイムで生成されます。
- **AIの対戦相手（「クイズ王」）**: ただ質問に答えるだけではありません。あなたは競争しています。AIも参加し、その正解率と「思考」行動はモデルによってシミュレートされます。
- **多言語サポート**: **英語**と**日本語**に完全対応（ブラウザ設定またはユーザー選択による自動検出）。
- **多彩なカテゴリー**: 一般知識、科学と自然、歴史、地理、映画とテレビ、音楽から選択できます。
- **レスポンシブデザイン**: Tailwind CSSで構築されたモダンなダークテーマのUIで、スムーズなアニメーションとトランジションを備えています。

## 🛠️ 技術スタック

- **フロントエンドフレームワーク**: React 19
- **言語**: TypeScript
- **スタイリング**: Tailwind CSS
- **AI統合**: [Google GenAI SDK](https://www.npmjs.com/package/@google/genai) (`@google/genai`)
- **モデル**: Gemini 2.5 Flash (速度と推論に最適化)

## 🤖 Gemini APIの活用方法

このプロジェクトでは、Google Gemini APIを2つの異なるコア機能に使用しています。

1.  **クイズ生成 (`generateQuiz`)**:
    -   `gemini-2.5-flash` と `responseSchema` を使用して、厳密なJSON出力を強制します。
    -   構造化されたトリビア問題（質問、4つの選択肢、正解）を生成し、解析とレンダリングを容易にします。

2.  **AIプレイヤーのシミュレーション (`getAiAnswer`)**:
    -   AIはペルソナ（「クイズ王」）として振る舞います。
    -   質問と選択肢を分析して回答を選択します。
    -   モデルは非常に高度ですが、間違いを犯すこともあり、競争がリアルで楽しいものになります。

## 🚀 セットアップとインストール

### 前提条件

- 有効な [Google Gemini API Key](https://aistudio.google.com/)。
- 最新のウェブブラウザ。

### 環境変数

アプリケーションを実行するには、環境変数で API キーを利用できるようにする必要があります。
ビルド環境または実行環境で `process.env.API_KEY` が設定されていることを確認してください。

### プロジェクトの実行

1.  **リポジトリをクローン** (該当する場合)。
2.  **依存関係をインストール**:
    ```bash
    npm install react react-dom @google/genai
    ```
3.  **開発サーバーを起動**。

*注: このプロジェクトは、特定のCDNベースの環境向けに `index.html` で importmap を使用して ES Modules を使用するように構成されていますが、標準的な React ソース構造として機能します。*

## 📂 プロジェクト構成

- **`App.tsx`**: メインアプリケーションロジックと状態管理（ゲームループ）。
- **`services/geminiService.ts`**: Google GenAI SDK とのすべてのやり取りを処理します（クイズ生成と回答の取得）。
- **`components/`**: 再利用可能な UI コンポーネント（QuizScreen, StartScreen, CategorySelection など）。
- **`i18n.tsx`**: 国際化を処理するためのコンテキストプロバイダー。
- **`constants.ts`**: ゲーム設定（カテゴリ、質問数）。
